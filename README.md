## 📝 Descrição do Exercício Desafio

###  Proposta:
Crie um array chamado alunos com 5 objetos;
Cada aluno deve ter:
nome;
idade;
disciplinaPreferida;

#### Código:
```
let alunos = [ 
    {nome: "Giovana", idade: 16, disciplinaPreferida: "Educação Física"},
    {nome: "Felipe", idade: 16, disciplinaPreferida: "Física"},
    {nome: "Antonella", idade: 16, disciplinaPreferida: "Quimica"},
    {nome: "Manuela", idade: 15, disciplinaPreferida: "Artes"},
    {nome: "Gbariel", idade: 16, disciplinaPreferida: "Matematica"},
];   
```
###   Desafio:
Usando os conhecimentos sobre estrutura de dados usando array e objetos, tente acessar:
o nome do primeiro aluno;
a disciplinaPreferida do último aluno;

#### Código:
```
alert (`O nome do primeiro aluno é : ${alunos[0].nome}`)
alert (`A disciplina preferida do último aluno (Gabriel) é: ${alunos[4].disciplinaPreferida}`)
```
